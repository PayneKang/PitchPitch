これは何？

声の高さ(ピッチ)で自機の高さ(ピッチ)を操作して障害物に当たらないように避けるゲームです。

中身の構成

PitchPitch.exe

動作環境

  • Windows Vista/7
  • Microsoft .NET Framework 2.0

インストール方法

  • フォルダ毎好きな場所に置く

アンインストール方法

  • フォルダごと消す
  • 設定ファイルは <ユーザディレクトリ>\AppData\Roaming\BeatDancer\> 以下にあるのでフォルダごと削除

遊び方

まず最初にオプションでオーディオ入力をするデバイスを選択する必要があります。
